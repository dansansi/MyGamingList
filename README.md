# MyGamingList
ASP.NET Core Web API for managing a personal game library (played, playing, wishlist), built with .NET, EF Core and SQLite.

## Tech Stack

- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- Identity Framework
- JWT Authentication
- Next.Js

## Purpose

This project is intended for users that wanna keep track of games played, and set each game's status accordingly (finished, wishlisted, dropped...) favorite, to through them easily.
